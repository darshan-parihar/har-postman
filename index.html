API Tools

HAR → Postman Converter

Drop a Chrome DevTools HAR export and get a ready-to-import Postman collection.

How to use
▼
  1. Open Chrome DevTools → Network tab → perform your API flow
  2. Right-click any request → Save all as HAR with content
  3. Enter your base URL and drop the HAR file below
  4. Click Generate → download both JSON files → import into Postman
Use Cases
Common scenarios for QA & testing teams
▼
Regression Testing Capture & replay a full API flow
WHENAfter a backend release — re-run the exact same API sequence to check nothing broke.
EXAMPLERecord login → fetch profile → update settings → logout. Drop HAR, generate collection, run in Postman. Compare responses to the previous run.
TIPEnable Remove duplicates to strip noise from page refreshes.
API Exploration Map an unfamiliar API without reading docs
WHENA new service is deployed and you need to understand its endpoints quickly.
EXAMPLEBrowse the app normally with DevTools open, save HAR. The generated collection shows every endpoint called — instant API map with real request & response bodies.
TIPEnable Organise folders to auto-group endpoints by feature area.
Auth Debugging Isolate authentication issues
WHENLogin works in the browser but fails in Postman or automation scripts.
EXAMPLECapture the login request in HAR. The generated collection has the HMAC pre-request script wired — open Postman Console to see the exact signature payload and compare it with what the browser sent.
TIPEnable Add test scripts to auto-capture the token from the login response into AUTH_TOKEN.
Bug Reporting Share a reproducible API scenario with devs
WHENReporting a bug that involves multiple chained API calls.
EXAMPLERecord the exact steps that trigger the bug. Generate collection. Share the JSON — developers can replay the exact sequence in one click with no manual setup.
TIPEnable Capture examples so the collection includes the actual buggy response body as reference evidence.
