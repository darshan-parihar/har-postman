HAR → Postman
Converter

Export HAR from Chrome DevTools, drop it here, get a ready-to-import Postman collection with HMAC signatures.

How to use
STEP 01
Open Chrome DevTools → Network tab → perform your API flow
STEP 02
Right-click any request → "Save all as HAR with content"
STEP 03
Enter your base URL, drop the HAR file, click Generate
STEP 04
Download the .json → Import into Postman → Set AUTH_TOKEN → Done
Configuration
Base URL(s) — first URL becomes {{BASE_URL}}, others are hardcoded
PRIMARY
HAR File
📂
Drop HAR file here
or click to browse — exported from Chrome DevTools
